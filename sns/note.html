Note.com連携

Note.com連携は、Note.comにて新しい記事が投稿された際に通知したり、記事を検索したりできます。

新しい記事が投稿された際に通知をする

以下は新しい記事が投稿された際に通知する機能をセットアップする方法です。

  1. /sns note add を入力し、ユーザー名を指定してから送信する
  2. /sns note listを入力し送信します。
  3. 通知したいユーザーのURLが表示されていれば成功です。

逆に記事通知を削除する場合は、/sns note removeで削除できます。
