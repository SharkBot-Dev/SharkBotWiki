# カスタムコマンド（Tag）

このBotでは、`/tag` スラッシュコマンドを使用してカスタムコマンド（Tag）の管理を行います。

### 1. カスタムコマンド（Tag）の新規作成
1. チャット欄で `/tag create 名前:<コマンド名>` を入力して実行します。
2. モーダル（入力画面）が表示されるので、以下の項目を入力します。
   * **コードを入力**: カスタムコマンド（Tag）が実行された時に出力される内容やスクリプト。
   * **説明を入力**: `/tag tags` で一覧を表示した際に表示される説明文。
3. 「送信」を押すと保存されます。

### 2. カスタムコマンド (Tag) の実行
作成したタグは、3通りの方法で呼び出せます。
* **スラッシュコマンド**: `/tag use コマンド名:<名前> 引数:<任意>`
* **メッセージ形式**: `(Prefix)<名前> <引数>`
    * 例: プレフィックスが `!.` でタグ名が `hello` の場合、 `!.hello` と入力。
* **カスタムスラッシュコマンド** (プレミアム版のみ): `/名前 引数:<任意>`
* **ボタンがクリックされたら** (プレミアム版のみ): ボタンのcustom_id

※カスタムコマンドの処理は以下の秒数以内に完了させる必要があります。
| 方法 | 秒数 |
| :--- | :--- |
| スラッシュコマンド (/tag use) | 5秒 |
| プレフィックス形式 (!.コマンド名) | 5秒 |
| カスタムスラッシュコマンド (/コマンド名) | 10秒 |

## Tagスクリプト構文ガイド

タグの「コード」内で使用できる特殊な構文一覧です。

### 基本変数（プレースホルダー）
実行時の状況に応じて自動的に置換されます。

| 構文 | 置換される内容 |
| :--- | :--- |
| `{args}` | コマンド実行時に渡された引数（メッセージ全体） |
| `{author}` | 実行者の表示名（ニックネーム） |
| `{author_id}` | 実行者のユーザーID |
| `{author_avatar}` | 実行者のアバター (URLではない) |
| `{author_avatar_url}` | 実行者のアバターURL |
| `{author_mention}` | 実行者のメンション |
| `{guild_name}` | サーバー名 |
| `{guild_icon}` | サーバーアイコン (URLではない) |
| `{guild_icon_url}` | サーバーアイコンURL |
| `{guild_id}` | サーバーID |
| `{count}` | サーバーの現在のメンバー数 |
| `{channel_name}` | チャンネル名 |
| `{channel_id}` | チャンネルID | 
| `{message_id}` | コマンドを呼び出したときのメッセージID |

### 変数操作 (`set` / `get`)
タグ内で一時的に値を保存・呼び出しできます。

* **作成**: `{set:変数名|値}` （この部分は出力時に消えます）
* **呼び出し**: `{get:変数名}`
    * 例: `{set:color|青}{get:color}い空` → `青い空`
    
---

### 関数 (ユーザー定義・マクロ)

スクリプト内で再利用可能な関数を定義し、後から呼び出すことができます。関数の実行時、引数はローカルスコープ（最優先スコープ）に格納されます。

#### 関数の定義 (`func`)

```text
{func:関数名 | 引数1,引数2,... | 関数の本体（プログラム）}

```

* 引数がない関数を定義する場合は、`{func:関数名 | 関数の本体}` と記述します。
* 定義された段階では、関数本体のプログラムは評価（実行）されません（遅延評価）。

#### 関数の呼び出し (`call`)

```text
{call:関数名 | 渡し引数1 | 渡し引数2 | ...}

```

#### 使用例

```text
{func:greet|name,emoji|こんにちは、{name}さん！{emoji}}

{call:greet|マイク|✨}

```

---

*出力: こんにちは、マイクさん！✨*

### 条件分岐 (`if` / `elif` / `else`)
条件に応じて出力するテキストを切り替えます。`elif:` や `else` を使った多角分岐にも対応しています。

#### 構文

```text
{if: 条件式 | 一致時のブロック
| elif: 条件式2 | 条件2一致時のブロック
| else | どの条件にも一致しない時のブロック}

```

#### 利用可能な比較演算子

条件式では以下の演算子が使用できます。数値比較（`>=`, `<=`, `>`, `<`）を行う際、値が数値に変換できない場合は自動的に条件不成立（`False`）扱いとなります。

| 演算子 | 説明 | 例 |
| --- | --- | --- |
| `==` | 左右の値が等しい | `{args} == 🍎` |
| `!=` | 左右の値が等しくない | `{status} != admin` |
| `>=` | 左辺が右辺以上（数値比較） | `{score} >= 80` |
| `<=` | 左辺が右辺以下（数値比較） | `{age} <= 19` |
| `>` | 左辺が右辺より大きい（数値比較） | `{hp} > 0` |
| `<` | 左辺が右辺より小さい（数値比較） | `{count} < 5` |

* **演算子がない場合**: `{if: {text}}` のように演算子を含めない場合、その文字列が空でなければ真（`True`）と判定されます。

#### 使用例

```text
{if: {score} >= 90 | **優秀です！**
| elif: {score} >= 60 | **合格です！**
| else | **がんばりましょう。** }

```

### ランダム・選択
* **数値ランダム**: `{random:最小~最大}` または `{random:最小,最大}`
    * 例: `運勢は {random:1~99} 点です`
* **要素選択**: `{choice:選択肢1|選択肢2|選択肢3}`
    * 例: `今日のランチは {choice:うどん|ラーメン|パスタ}`

---

### 特殊機能
* **URLエンコード**: `{urlencode:文字列}`
    * 文字列をURLで使用可能な形式に変換します。
* **ロール付与**: `{addrole:ロールID}`
    * 実行したユーザーに指定したIDのロールを付与します。（この構文自体は出力時に消えます）
* **ロール剝奪**: `{removerole:ロールID}`
    * 実行したユーザーに指定したIDのロールを付与します。（この構文自体は出力時に消えます）
    
### リアクション
* **リアクション追加**: `{addreaction:チャンネルID|メッセージID|絵文字}`
    * 実行したチャンネルのメッセージに指定した絵文字をリアクションをします（この構文自体は出力時に消えます）
* **リアクション削除**: `{removereaction:チャンネルID|メッセージID|リアクションを追加した人|絵文字}`
    * 実行したチャンネルのメッセージから指定したリアクションを削除します（この構文自体は出力時に消えます）

### 埋め込み
* **埋め込みの追加**: {embed:コード}
		* 埋め込みをタグメッセージに追加します。最大10個まで追加できます。
例: `{embed:title=タイトル,description=説明,image=画像url,thumbnail=サムネイル画像url,color=数字の色(16711680),footer=フッターテキスト,footer_icon=フッターアイコンURL,author=ユーザーテキスト,author_icon=ユーザーアイコンURL}`

### コンポーネント（プレミアム版専用）
* **ボタンの追加**: {button:コード}
		* ボタンをタグメッセージに追加します。追加できる数には制限があります。
例: `{button:label=ボタンラベル,custom_id=カスタムID,style=ボタンのスタイル(例:danger),url=ボタンのURL(styleや、custom_idと一緒に指定できません。,emoji=絵文字)}`

## Tagスクリプト記述例

**例：おみくじコマンド**
```text
{author_mention}さんの今日の運勢は...
**{choice:大吉|中吉|小吉|末吉|凶}** です！
(ラッキーナンバー: {random:1~99})
```

**例：引数によって反応を変える**
```text
{if: {args} == 🍎 | **りんご**だね！ 
| elif: {args} == 🍏 | **青りんご**だね！ 
| elif: {args} == 🍙 | **おにぎり**だね！食べたいな！
| else | なんだろう、わからないな }
```

**例: さいころコマンド**
```text
{func:dice | {random:1~6}}

🎲 {call:dice}
```
