基本設定

SharkBotでは基本的な設定が存在します。このページでは基本的な設定の説明が書いてあります。

ダッシュボードにログインする

SharkBotには一部機能を直感的に設定できる、ダッシュボードが存在します。

ダッシュボードには以下の手順でログインすることができます。

  1. ダッシュボードのページ にアクセスし、「Discordでログイン」をクリックしてログインを行います。
  2. サーバーの一覧が表示されます。
  3. 設定したいサーバーを選択します。
  4. サーバー設定のトップページが表示されれば成功です。
    ※導入画面が表示された場合は導入を行ってください。

Prefix（頭文字）を設定する

SharkBotでは「!.」で始まるコマンドにて「!.」以外の文字列を使用できるように設定することもできます。
Prefix（頭文字）は以下の手順で設定することができます。

※Prefixには一文字の記号を設定することを推奨します。例: !、?

  1. /settings prefixと入力し、Prefixを指定して送信する
  2. 「[指定したPrefix]コマンド名 引数1 引数2 ..」という感じで送信し、Botが反応すれば成功です。

不要なコマンドを無効にする

SharkBotでは、不要なコマンドをダッシュボードにて無効にすることが可能です。
コマンドを無効にするには、以下の手順で無効にすることができます。

  1. ダッシュボードにアクセスする
  2. 「コマンド一覧」を選択し、無効にしたいコマンドを検索します。
  3. トグルスイッチをクリックし、オフにします。
  4. 「設定を保存する」を押して保存します。
  5. Discordにて、無効にしたコマンドを実行し、無効になっているというメッセージが表示されていれば成功です。

アバターのデコレーションを設定する

SharkBotでは、Botのアバターにサーバーごとのデコレーションを設定することができます。
デコレーションを設定するには、以下の手順で設定できます。

  1. /bot decorationと入力し、送信する
  2. デコレーションのプレビューとデコレーション選択画面が表示されるため、選択する
  3. デコレーションのプレビューが表示され、気に入ったら、保存ボタンを押す。
  4. サーバーごとのアバターにデコレーションが適用されます。
  5. あとはデコレーションが適用されたBotを楽しむだけです！

※デコレーションを外す場合は、1.の手順を実行後に、「なし」を選択してから保存をしてください。
